Plugin Installation

  1. Download the plugin using the button above
  2. Go to your WordPress dashboard → Plugins → Add New → Upload Plugin
  3. Choose the downloaded file and click "Install Now"
  4. After installation, click "Activate Plugin"
  5. Go to Settings → Water Report to configure the plugin
What the plugin does:
  • Creates a [water_report_form] shortcode you can use anywhere on your site
  • Automatically loads the necessary JavaScript and CSS
  • Adds a settings page where you can configure the server URL
  • Properly handles script loading to avoid conflicts with other plugins
  • Only loads scripts on pages where the shortcode is actually used

Manual Embedding (Alternative Method)

If you prefer not to use a plugin, you can manually embed the form using these steps:

Step 1: Add the Script to WordPress

Add the following code to your WordPress site's header or footer (via theme settings, a header/footer plugin, or in your child theme):

<script> window.WATER_REPORT_API_URL = '//www.waterreportapp.com'; </script> <script src="//www.waterreportapp.com/water-report-embed.js"></script>

Step 2: Add the Embed Container

Place this div anywhere you want the form to appear in your WordPress content:

<div data-water-report-embed></div>

You can add this to:

  • WordPress pages or posts using the HTML/Custom HTML block
  • WordPress widgets that accept HTML
  • Theme template files

Step 3: Creating a WordPress Shortcode (Optional)

For easier embedding, you can create a shortcode by adding this code to your theme's functions.php file or a custom plugin:

function water_report_form_shortcode() { return '<div data-water-report-embed></div>'; } add_shortcode('water_report_form', 'water_report_form_shortcode');

Then use it in your content with: [water_report_form]

Important: The form supports both HTTP and HTTPS using protocol-relative URLs (//www.waterreportapp.com). It will automatically use the same protocol as your WordPress site.
